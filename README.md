## 📖 使用教學

### 新增記錄
1. 點擊「新增記錄」卡片
2. 選擇月經開始日期（只能選今天或以前）
3. 點擊「新增日期」

### 查看統計
自動顯示（至少 2 筆記錄）：
- **平均週期**：整數天數
- **距上次**：距離上次月經的天數
- **週期數**：已記錄的完整週期數
- **穩定度**：穩定 / 普通 / 不穩定

### 預測功能
- 根據歷史平均週期預測下次月經日期
- 顯示「還有 N 天 / 預計今天 / 已超過 N 天」倒數
- 跨年預測會顯示完整年份

### 記錄管理
- 記錄以最新為優先排列
- 每筆顯示距上次的間隔天數（最舊的一筆顯示「首筆記錄」）
- 每筆都可以個別刪除

### 備份資料
- **匯出**：點「匯出」→ 產生 JSON 檔
- **匯入**：點「匯入」→ 選擇備份檔（自動略過無效日期）
- **清除**：點「清除」→ 刪除所有資料

### 核心演算法

#### 本地日期解析（避免時區偏移）
```javascript
function parseLocalDate(str) {
  const [y, m, d] = str.split('-').map(Number);
  return new Date(y, m - 1, d); // 以本地時間建立，不受時區影響
}
```

#### 平均週期計算
```javascript
const cycles = [];
for (let i = 1; i < sorted.length; i++) {
  const diff = (parseLocalDate(sorted[i]) - parseLocalDate(sorted[i-1])) / (1000*60*60*24);
  cycles.push(diff);
}
const avg = cycles.reduce((a, b) => a + b, 0) / cycles.length;
const avgRounded = Math.round(avg);
```

#### 穩定度評估
```javascript
const variance = cycles.reduce((sum, c) => sum + Math.pow(c - avg, 2), 0) / cycles.length;
const stdDev = Math.sqrt(variance);
// stdDev < 3: 穩定
// stdDev < 7: 普通
// stdDev >= 7: 不穩定
```
